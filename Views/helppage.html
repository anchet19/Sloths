  • What is the purpose of this software?
  • This software handles the process of scheduling time slots automatically so the administrator does not need to do it by hand.
  • Who is this software for?
  • This software is for the employees of ASRC Federal.
  • What are the roles for this software?
  • This software supports the roles of administrator, manager, and user.
  • What do the colors mean on the Calendar page?
  • Please take a look at the Legend page for more information.
  • What happens if I request a time slot at the time time as another employee?
  • No problem! When the administrator finalizes the schedule, our fairness algorithm calculates which employee gets the time slot.
  • I did not get the time slot I requested and it was given to another employee (or employees) that also requested it. Is there a "secret" to always get a slot?
  • Not precisely. We try to make it as fair as possible for all employees. Every user has a "score" that changes depending on that respective user's choices when selecting time slots. This score is used as a "tiebreaker" when multiple users request a time slot. Don't worry, if you did not get a slot you requested, this increases your chance to win a tiebreaker next time.
  • What version is this software?
  • At the time this software is presented to ASRC Federal in December, the software is at version 1.0.
